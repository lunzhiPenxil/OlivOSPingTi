# OlivOSPingTi

基于[OlivOS](https://github.com/OlivOS-Team/OlivOS)和[最佳平替](https://www.pingti.xyz/)的平替查询插件。

## 指令
> `/` `平替` `查询内容`  
`/` `pingti` `查询内容`  
`/` `pt` `查询内容`  
`.` `平替` `查询内容`  
`.` `pingti` `查询内容`  
`.` `pt` `查询内容`  

### 说明
> 基于[最佳平替](https://www.pingti.xyz/)网站的接口实现，如作者认为存在滥用请联系我将本插件进行删库
